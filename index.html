Privacy Policy — SpacePath

SpacePath собирает только необходимые данные для работы приложения:

  • Дата рождения и знак зодиака (для персонализации прогнозов)
  • Настройки пользователя и записи дневника
  • Анонимные данные об использовании приложения (через Firebase Analytics)

Использование данных:

  • Для генерации персонализированных гороскопов через AI (Google Gemini)
  • Для хранения и отображения информации в приложении
  • Для аналитики и улучшения работы приложения

Обмен данными с третьими лицами:

  • Ваши личные данные не продаются и не передаются сторонним компаниям
  • AI (Gemini) получает данные только для генерации ответов внутри приложения

Безопасность: мы используем стандартные меры безопасности Firebase и Android для защиты данных

По вопросам конфиденциальности пишите на: support@spacepath.app
